# Networker

A simple CLI to monitor your local network. Navigate to the repo directory and utilize the `networker` bash script for easy access. Try `./networker --help` to get started!

> **Note:** The CLI requires admin privileges so `sudo` is utilized in the bash script.

## Initial Commands
 - Utilize the `./networker init` command to seed the sqlite database with port reference data.
 - Utilize the `./networker network init` command to scan and log network and device information.
